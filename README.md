## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```


run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [https://portal-indyco-next.5p7m7t.easypanel.host](https://portal-indyco-next.5p7m7t.easypanel.host) with your browser to see the result.
